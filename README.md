### Golems — MCP servers for the agentic IDE era

Three open-source MCP servers. 44 tools. Works with Claude Code, Codex, Gemini, Cursor, and Kiro.

| Project | What it does | Install |
|---------|-------------|---------|
| [**BrainLayer**](https://brainlayer.etanheyman.com) | Persistent memory for AI agents — knowledge graph, hybrid search, 12 MCP tools | `pip install brainlayer` |
| [**VoiceLayer**](https://voicelayer.etanheyman.com) | Voice I/O for AI agents — local STT via whisper.cpp, neural TTS, 10 MCP tools | `npm i voicelayer-mcp` |
| [**cmuxLayer**](https://cmuxlayer.etanheyman.com) | Terminal orchestration — spawn panes, read screens, coordinate agents, 22 MCP tools | `npm i -g cmuxlayer` |

Each ships with a native macOS companion: **BrainBar** (menu bar memory dashboard) and **VoiceBar** (always-on voice interface).

Built on the [Golems](https://github.com/EtanHey/golems) agent ecosystem — 55+ skills, autonomous coding loop, 6 MCP servers.
